# スプレッドシート構成（たなかんち2 予約システム）

Google スプレッドシートを新規作成し、以下5つのシートを準備してください。
**1行目はヘッダ**（そのままコピーして貼り付け可）。

---

## シート1: `予約`

予約データの本体。GASが自動で追記します。

| A | B | C | D | E | F | G | H | I | J | K | L | M |
|---|---|---|---|---|---|---|---|---|---|---|---|---|
| 予約ID | LINE_UID | 表示名 | 電話番号 | 来店日 | 開始時刻 | 終了時刻 | 人数 | 席種 | 席ID | コース | ステータス | 作成日時 |

ヘッダ貼り付け用（タブ区切り）:
```
予約ID	LINE_UID	表示名	電話番号	来店日	開始時刻	終了時刻	人数	席種	席ID	コース	ステータス	作成日時
```

- **席種**: `カウンター` / `テーブル` / `個室`
- **コース**: `席のみ` / `飲み放題`
- **ステータス**: `確定` / `キャンセル`
- 来店日は `YYYY-MM-DD`、時刻は `HH:mm` 形式で保存

---

## シート2: `席マスタ`

席の一覧。最初に手入力して以後は変更しません。

| A | B | C |
|---|---|---|
| 席ID | 種別 | 定員 |
| C1 | カウンター | 1 |
| C2 | カウンター | 1 |
| C3 | カウンター | 1 |
| C4 | カウンター | 1 |
| C5 | カウンター | 1 |
| C6 | カウンター | 1 |
| T1 | テーブル | 4 |
| T2 | テーブル | 4 |
| T3 | テーブル | 4 |
| P1 | 個室 | 4 |

ヘッダ貼り付け用:
```
席ID	種別	定員
```

---

## シート3: `顧客`

LINEから予約したお客様の自動台帳。

| A | B | C | D | E |
|---|---|---|---|---|
| LINE_UID | 表示名 | 電話番号 | 来店回数 | 最終来店日 |

ヘッダ貼り付け用:
```
LINE_UID	表示名	電話番号	来店回数	最終来店日
```

---

## シート4: `営業日例外`

大型連休・臨時休業・特別営業時間の設定用。

| A | B | C | D |
|---|---|---|---|
| 日付 | 区分 | 開店 | 閉店 |

ヘッダ貼り付け用:
```
日付	区分	開店	閉店
```

- **区分**: `休業` / `営業`
- 区分が `営業` の場合は開店・閉店時刻を上書き（HH:mm）
- 区分が `休業` の場合はC・D列は空欄でOK

例:
```
2026-05-03	休業
2026-12-31	営業	17:00	21:00
```

---

## シート5: `設定`

GASが参照する各種設定値。キー=値の2列構造。

| A | B |
|---|---|
| キー | 値 |
| 店名 | たなかんち2 |
| 電話番号 | 03-xxxx-xxxx |
| 通常開店 | 17:00 |
| 通常閉店 | 23:00 |
| ラストオーダー | 22:00 |
| 滞在時間分 | 120 |
| 最終予約開始 | 20:00 |
| 飲み放題価格 | 6000 |
| 受付可能日数 | 90 |
| キャンセル受付期限日数 | 3 |
| 自動受付上限人数 | 4 |
| 貸切人数下限 | 15 |
| LIFF_URL | https://liff.line.me/xxxxxxxxx-xxxxxxxx |
| OWNER_LINE_UID | （店主のLINE userId） |

ヘッダ貼り付け用:
```
キー	値
```

> `LIFF_URL` と `OWNER_LINE_UID` は後で記入します。

---

## セットアップ手順

1. Google スプレッドシート新規作成 → ファイル名を「たなかんち2_予約管理」に
2. シート名を上記5つに合わせる（左下の `+` で追加、シートタブを右クリック → 名前を変更）
3. 各シートのヘッダ行と席マスタ・設定の初期値を入力
4. URLバーから **スプレッドシートID** をコピー（`/d/`と`/edit`の間の文字列）
   - これは後でGASのスクリプトプロパティに登録します
